Lancer le programme:

Ouvrir le fichier PhysicalEngineTeam1.sln avec Visual Studio puis compiler la solution.
A noter que le dossier du projet doit être placé dans le dossier of_v0.12.0_vs_release\apps\myApps pour éviter les erreurs de dépendances.


COMMANDES DE BASE:

	- ZQSD : diriger la particule du blob contrôlée (clavier AZERTY)
	- B : spawn/despawn le blob
	- H : déplacer le sol
	- L : Split le blob
	- O : rassembler le blob
	- K/J : bouger la caméra
	- M: activer la caméra trackant le blob
	- ESPACE : démo de particule simple
	- X : démo de ressort
	- C : démo de câble
	- V : démo de tige

Lancer les tests:

Exécuter le fichier test.sln dans le dossier /test
