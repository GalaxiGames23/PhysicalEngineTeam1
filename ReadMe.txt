Lancer le programme:

Ouvrir le fichier PhysicalEngineTeam1.sln avec Visual Studio puis compiler la solution.
A noter que le dossier du projet doit être placé dans le dossier of_v0.12.0_vs_release\apps\myApps pour éviter les erreurs de dépendances.


COMMANDES DE BASE:

	- t : montrer ou cacher la trace
	- r : reset la trace
	- m : activer ou désactiver le suivi de la caméra 
	- k/j : modifier le zoom de la caméra
	- espace/c/v/b/n : lancer une démo
	- e + souris : faire tourner la caméra autour de la boite
	- y : activer ou désactiver l'affichage de l'octree

Lancer les tests:

Exécuter le fichier test.sln dans le dossier /test
